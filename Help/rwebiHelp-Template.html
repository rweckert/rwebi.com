Toolbar Menu Options:
Toggle Menu Options:
Opens a menu of options along the top right side of the text or data area with more options and features for your project. Each of these options are described below under "Toggle Menu Options".
File Menu:
Opens a genaral menu pertaining to features related to opening, saving and other features related to the application or module. The File Menu is described in detail below under "File Menu".
Open File:
Open file dialog window will appear allowing you to locate and and open a text file on your local device for editing.
Cut Selected:
Cut selected text into the clipboard.
Copy Selected:
Copy selected text into the clipboard.
Paste Selected:
Paste selected text into the clipboard. Text will be inserted at the current location of the cursor in the text document.
Toggle Menu Options:
New File:
Creates a new document file. You will be prompted to confirm creating a new file.
Increase Font:
Increases the font size in the document.
Decrease Font:
Decreases the font size in the document.
 Toggle Editing Lock:
Locks the document from any further changes or editing. Once the lock icon is selected, a message will appear stating that the document has been locked. The lock icon will change from a "unlocked" state to a "locked" state to indicate that the document has been locked. To unlock the document, simply click on the "locked" icon and a message will appear stating that the document has been unlocked allowing further changes and editing. By default the lock icon is in the "unlocked" state.
 Toggle Spell Checking:
Enables browser controlled spell checking. By default this is disabled and represented with a purple background inside the icon. Clicking on this icon will enable spell checking the current text document. Spelling and punctuation errors are located and defined by your browser and are generally underlined to help identify. When spell checking is enabled, the icon will change to have a green background and any indications of spelling or punctuation errors will not be visible.
 Toggle Word Wrap:
Long line wrapping of the current document. By default word wrapping is enabled. Selecting the word wrapping icon will change the viewing area of the text to not wrap long lines. When word wrapping is enabled, the icon will change to indicate "long lines", and when disabled the icon will change back to its default enabled state.
Help About Settings:
Help/About/Settings:
Opens the information view and provides help information regarding the application or module.
Help:
Opens the 'Help' view and provides help information regarding the application or module.
About:
Opens the 'About' view and provides general application information regarding the application or module.
Share Link:
Copies a link of the application or module to clipboard memory. A message will appear displaying the link address and stating that the link has been copied to the local clipboard.
Mode:
Allows to change the theme of the application or module. Two options for "Light Mode" or "Dark Mode" will over-ride the current theme when selected.
Default Theme:
The default "Light Mode" is used unless the users browser dark color scheme preference has been enabled. If the browsers current setting is to the "dark" color scheme, then the "Dark Theme" is used.

Text Stats:
Shows text statistics of your text or data view. As you use the text area for either data or text, changes will be shown in the text statistics area by total words, characters, text, and lines. These are described as:

Words: Is counted by the space character between text. A long line with 255 characters would be considered as one word.
Characters: Counts all characters of the current document including space characters and line breaks.
Text: Counts all text of the current document including the space character. The text count does not include line breaks.
Lines: Shows a total of lines in the document, and does count empty lines as part of the total.
Viewing Area / Editor Workspace:
Default Limits: 512 KB or 524,288 characters.
Data Type: The default data type and limit is measured in UTF-16 code units.

Default Limits: While there is no strict HTML specification limit, practical constraints depend on user agent (browser) limitations and the maxlength attributes:

- 524,288 characters
- 524,288 characters = 524,288 bytes
- 524,288 bytes \ 1,024 bytes/KB = 512 KB

Note: There is no maximum length set for any of the text or data area inputs in rwebi, please keep in mind that large amounts of text or data may cause local browser performance issues when processing.
